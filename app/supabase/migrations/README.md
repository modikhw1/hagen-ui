# Legacy migrations

Den har mappen ar ett aldre migrationsspar och ar inte langre canonical.

Gor inte nya schemaandringar har och kor inte dessa filer som primar migrationskedja i nya miljoer.

Canonical migrations finns i:

- `supabase/migrations`

Den kedjan ar den som ska matcha aktuell adminv3-, RBAC-, Stripe- och TikTok-modell.

Den har legacy-mappen finns bara kvar som historiskt underlag medan repot fortfarande stadas upp.
