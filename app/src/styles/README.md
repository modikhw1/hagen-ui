# Admin Styles

`admin-tokens.css` samlar admin-domänens designtokens utan att ändra den visuella renderingen.

- Färger ligger som `--admin-color-*` och refererar till de semantiska app-variablerna.
- Radius, spacing, typografi och skuggor ligger som `--admin-radius-*`, `--admin-space-*`, `--admin-font-*` och `--admin-shadow-*`.
- `data-admin-theme="dark"` finns endast för intern preview/styleguide och påverkar inte befintliga vyer förrän attributet används.
