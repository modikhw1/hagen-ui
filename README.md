# hagen-ui

> MVP frontend for the TikTok concept marketplace.

## Quick Start

```bash
cd app
npm install
npm run dev
```

Visit http://localhost:3000

## What This Is

A simple MVP to validate the core idea: helping small businesses find proven TikTok concepts to recreate.

## Routes

| Route | Description |
|-------|-------------|
| `/` | Landing page |
| `/start` | Onboarding flow |
| `/dashboard-1` | Dashboard v1 |
| `/dashboard-2` | Dashboard v2 |
| `/concept/[id]` | Concept detail |
| `/profile` | User profile |

## Tech Stack

- Next.js 16 + React 19
- Mantine UI v8
- Tailwind CSS v4
- TypeScript
